## Who am i?
**I am Cheetah, a teenager that loves coding and has been coding since i was 7 years old.**

## My projects
### Web projects
- [pesto](https://pesto.ch0.dev) - A simple but useful pastebin clone, written in rust to learn SQL and SQLite
- [wrix](https://wrix.ch0.dev) - A chat room, written in rust to learn websockets
- [kiln](https://kiln.ch0.dev) - A url shortener, written to learn the TailwindCSS standalone CLI
### Software projects
- [Raytracer](https://github.com/CheetahDoesStuff/Raytracer) - A raytracer written in rust, with features such as quads and procedural textures.
- [xv0](https://github.com/CheetahDoesStuff/xv0) - A simple operating system written in rust, that boots, and runs doom!
- [coolcaves](https://github.com/CheetahDoesStuff/coolcaves) - A tui that procedurally generates caves! My first rust project.
- [dirlink](https://github.com/CheetahDoesStuff/DirLink) - A cli tool written in python to navigate your filesystem faster.
- [HolyShit](https://github.com/CheetahDoesStuff/HolyShit-Esolang) - An interpreted esoteric programming language written in python, similar to brainf*ck but with some extra features and even more annoying!
- [PyCodeObfuscator](https://github.com/CheetahDoesStuff/PyCodeObfuscator) - A tool to obfuscate python code using recursive chunked, randomly encrypted blocks of code.
- [BEANS](https://github.com/CheetahDoesStuff/BEANS) - An interpreted programming language with assembly-like syntax and a GUI debugger which shows live memory updates etc.
